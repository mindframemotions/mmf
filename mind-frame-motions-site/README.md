# Mind Frame Motions – GitHub Pages

Multi‑Page‑Setup für deine Immobilien‑Videografie (kostenlos via GitHub Pages).

## Seiten
- **Startseite** (`index.html`)
- **Listings** (`listings.html`)
- **Preise** (`preise.html`)
- **Über mich** (`ueber-mich.html`)
- **Impressum**, **Datenschutz**

## Deploy
1. Neues **öffentliches Repository** erstellen (z. B. `mfm`).
2. Dateien ins **Repo‑Root** laden.
3. **Settings → Pages** → Deploy from a branch → `main` / `/`.
4. Ersetze in `sitemap.xml` & `robots.txt` `USERNAME/REPO` bzw. Domain.

## Anpassen
- YouTube‑IDs in `listings.html` austauschen.
- Adresse im **Impressum** ergänzen (Pflicht in DE).
- Farben/Branding in `styles.css`.
